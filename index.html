DeepWiki Module Documentation Generator

为 C++ 模块自动生成架构可视化、API 完整性、可运行代码示例的技术文档。 同时输出 Markdown 和 HTML 两种格式，单文件零依赖，离线可用。

⚙ Core Features

六大核心特性，覆盖从生成到校验的完整工作流。

▦

架构可视化

内联 SVG 图表（架构图、状态机、流程图），支持点击放大、滚轮缩放、拖拽平移。所有颜色使用 CSS 变量，暗色/浅色主题自动适配。

☰

API 参考

Tabs 组件展示多类 API，完整函数签名、参数说明、返回值。每个函数附带源码行号引用，可直接跳转到实现。

♼

双格式输出

同时生成 .md 和 .html 两种格式。Markdown 用于 Git 版本控制和 Code Review，HTML 用于本地浏览和团队共享。

◑

暗色/浅色主题

自动跟随系统偏好（prefers-color-scheme），一键切换，localStorage 持久化。所有组件、图表、代码块均适配双主题。

✓

代码高亮

highlight.js 自动语法高亮 + 一键复制按钮。支持 C++、JSON、Bash 等语言。零依赖后备方案：手动 CSS 类着色。

⚠

闭环校验

11 类内容自动校验 + 修复：Mermaid 语法、章节 ID、代码块、表格、TOC 完整性、源码引用、术语表、Scope 声明等。校验-修复-复查闭环。

▶ Quick Start

三步完成模块文档生成。

安装

将 skill 文件放到 ~/.claude/skills/deepwiki-module-skill/ 目录

使用

在 Claude Code 中说 "给 XXX 生成 deepwiki 文档"

校验

自动生成后运行 validate-doc.js 闭环校验

ⓘ
Tip 生成的文档同时输出 .md 和 .html 两种格式到 doc/tech-docs/ 目录。Markdown 用于 Git 版本控制，HTML 用于本地浏览。 
# 在 Claude Code 中触发文档生成
给 Task 生成 deepwiki 文档

# 或使用英文
generate deepwiki doc for CycleScheduler

# 校验生成的文档
node ~/.claude/skills/shared-docs/validate-doc.js --new-doc --fix --type module doc/tech-docs/Task_Design.html
☷ Technical Specs

DeepWiki 的技术规格与能力矩阵。

FeatureDescription
Output Format .md + .html（单文件零依赖，可离线浏览）
Diagrams 内联 SVG（架构图、状态机、流程图），使用 CSS 变量适配主题
Interaction 点击放大（Fullscreen）、滚轮缩放、拖拽平移、Esc 关闭
Theme 暗色/浅色双主题，跟随系统偏好，localStorage 持久化
Validation 11 类闭环校验（Mermaid / ID / 代码块 / 表格 / TOC / 源码引用 / 术语表 / Scope 等）
Language 中文/英文模板（template.md / template.en.md）
Chapter Structure 12 章节（概述、设计目标、架构、核心概念、状态机、流程图、实现细节、API 参考、使用指南、FAQ、测试覆盖、附录）
Code Highlight highlight.js + 手动 CSS 着色后备 + 一键复制按钮
Collapsible Sections <details> / <summary> 可折叠章节，重要章节默认展开
Source References 每个 API 附带 file:line 源码行号引用，可直接跳转
TOC 自动生成侧边栏目录，ScrollSpy 高亮当前章节
Print 打印优化，自动隐藏导航和交互元素
✓
Zero Dependencies 生成的 HTML 是完全独立的单文件，所有 CSS 和 JavaScript 均内嵌。唯一的外部引用是 highlight.js CDN（可选，有 CSS 后备方案）。
